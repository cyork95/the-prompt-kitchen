How to get your GitHub token (one-time setup)

  1. Go to github.com/settings/tokens → Fine-grained tokens
  2. Click "Generate new token"
  3. Name it: Prompt Kitchen Publisher, set expiration to 1 year
  4. Under Repository access: select Only select repositories → pick the-prompt-kitchen
  5. Under Permissions → Repository permissions: set Contents to Read and write
  6. Click Generate token, copy it, paste it below

Your token is stored only in this browser (localStorage). It never leaves your device except to call the GitHub API directly.

How this tester works

Step 1 — Choose a prompt from the library (research, creative, coding, etc.) or write your own. Step 2 — Pick which AI tools to test (or let the tester choose randomly). Step 3 — Open each tool with a direct link, paste the prompt, and mark it done. Step 4 — Paste the AI responses back here. Step 5 — Export formatted HTML to add to the Tasting Menu, or JSON to store elsewhere.

Sessions are auto-saved to your browser (localStorage) so you can close and come back. No account required. Nothing leaves your browser.
